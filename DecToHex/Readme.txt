*Requirement:
Linux, SDK Android 22, Java 7

*What are included in this project:

1. A simple program to convert an Integer 32bits input to a hexadecimal String representation (ConvertDec.java). 

2. Build an android program UI with a text field input and a button(MyActivity.java).

3. The text field will be used to input the integer. The button will be used to call and handle the process to convert.

4. The result will be displayed in a pop up and via log simultaneously. 

*How to use:

run the bash script make.sh, it will 

1. generate R.java file

2. compile and generate dex files

3. generate apk file

4. generate java docs
